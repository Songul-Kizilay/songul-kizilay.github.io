# Operator Snapshot

Batman University Computer Engineering graduate with a working style shaped by both attacker curiosity and defensive visibility. My current focus is manual web application pentesting, API security, bug bounty validation, and turning raw findings into usable technical notes.

Enforsec gave me direct red team exposure, while Infinitum IT and the Ministry of Interior gave me enough SOC and defensive visibility to understand how attacker behavior gets seen on the other side. That context matters because I do not stop at finding the bug; I also think about logs, telemetry, detection gaps, and operational impact.

BITES, Deneyap Türkiye, and CyberHero Academy added monitoring, automation, lab-building, and mentoring experience. The result is a profile built on hands-on security work rather than portfolio aesthetics alone.
